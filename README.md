## Overview
CineMatch is a Python-based application that movie ticket bookings. It allows users to browse movies, check real-time seat availability, and book tickets, with all data stored persistently in text files.

## Features
* **View Movies:** Browse titles, prices (in Rs.), and available seats.
* **Smart Booking:** Prevents booking if seats are sold out.
* **Dynamic Data:** Updates seat counts instantly after every booking.
* **Booking History:** Keeps a log of all transactions.
* **Modular Design:** Code is split into logic, storage, and UI modules.

## Technologies Used
* **Language:** Python 3.1
* **Storage:** simple File Handling (Text-based persistence)
* **Tools:** Git & GitHub

## How to Run
1.  Clone the repository.
2.  Run `python main.py` in your terminal.
3.  Follow the on-screen menu.

## Instructions for testing
1.  Check if the movies are displayed correctly.
2.  Check for error handling, (try entering invalid ID).
3.  Check for transaction history.
